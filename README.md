# javaEE 环境一键配置脚本
如果你刚开始学习javaEE又想懒人搞定环境搭建（ai还是太好用了）<br>
本脚本一键配置 jdk + maven + jetty 并提供了命令行来简化创建标准项目和运行流程。

## 如何安装
此为linux脚本！！！ windows请使用wsl <br>

1.找到适合你系统版本的.sh文件下载到本机 <br>

输入 `sudo chmod +x 你刚刚下好的文件` 添加权限 <br>

2.输入 `sudo ./你刚刚下好的文件 `  开始运行安装程序

## 如何使用

仅搭建标准项目： `web -g 包名 -p 项目名` <br>
搭建标准项目并开始运行 `web -g 包名 -p 项目名 -r` <br>
运行项目 `web -r 项目文件夹`
